The format is simple: [filename]?[lineNum[-endLineNum];], ...

This line in your HTML...


<pre><code data-srcbyline='../srcByLine.js?1-11;162-164'></code></pre>

...produces this block (we have some basic styles attached)

And this line in your HTML...


<pre><code data-srcbyline='../srcByLine.js?1-12;47;49;57;59;85;87;108;110;125;127;142;144;160-164'></code></pre>

...produces this block

Note that files are cached on each page load, so although these two examples are on the same page, the file will only be retrieved once.
